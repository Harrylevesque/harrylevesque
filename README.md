2024-2025
My 2024-2025 science fair project was an artificial intelligence that could detect faild 3d prints and stop the printer in real time to allow a human to go check if it was truly failed or not. It was quite fun learning Python and TensorFlow to make something happen in real time in front of my eyes. 2025-2026
My 2025-2026 science fair project was a backend system for a new login protocol that could not be used to fish data, MITM safe and PQC safe. The goal of the project was :
